# utils

Auto-imported plain JavaScript helpers for the app.


## Contents

* [wallpaper.js](wallpaper.js) - the pure canvas renderer. It exports `DEVICE_SIZES`, `THEMES`, and `renderWallpaper`, which paints the wallpaper onto a `<canvas>` at full iPhone resolution.

Phase 1 of [docs/technical-plan.md](../../docs/technical-plan.md) adds `backgrounds.js`, the curated photo background manifest and loader, to this folder.
