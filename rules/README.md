# Rules

This folder contains rules for AI agents to follow when working in this repository.
Each rule is a Markdown file that states one convention or constraint in plain language.
Agents read these rules alongside [AGENTS.md](../AGENTS.md) to keep tickets, specs, and notes consistent.


## Usage

To add a rule, create a Markdown file in this folder (for example `ticket-format.md`) that states the rule and when it applies.
Keep each rule focused on one topic so agents can load only the rules that match their task.
