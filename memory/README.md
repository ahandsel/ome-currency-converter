# Memory

This folder contains project-scoped memory for this repository.
Memory files are used to store reference content, feedback, and other information that is relevant to the project and should be easily accessible to contributors.


## Usage

To add new memory content, create a new markdown file in this folder with a descriptive name and description in the front matter.

Update the [MEMORY.md][] file to include a link to the new memory file, along with a brief description of its content.

[MEMORY.md]: ../MEMORY.md
