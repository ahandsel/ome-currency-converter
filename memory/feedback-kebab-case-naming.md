# File and folder naming convention

Use kebab-case for folder names and file names.

**Why:** User explicitly requested kebab-case as the naming convention.

**How to apply:** When creating or renaming folders and files, use kebab-case by default. This includes project folders and new repo files unless an exception below applies.

**Exceptions:**

* Special files like `README.md`, `MEMORY.md`, and `.claude` folder contents are exempt from this rule to maintain consistency with common conventions and existing structure.
