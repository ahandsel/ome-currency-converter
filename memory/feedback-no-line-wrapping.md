# No sentence line-wrapping

Do not split a sentence into multiple lines (soft-wrapping) in script headers, comments, or usage text.

**Why:** The user finds line-wrapped sentences hard to read.

**How to apply:** When writing or editing documentation, comments, version history entries, or help text in scripts, keep each sentence or logical entry on a single line regardless of length.
