# Commands

This folder contains slash-command definitions for this repository.
Each command is a Markdown file that defines a reusable AI workflow you invoke by name.


## Usage

To run a command, type its name with the appropriate prefix for your AI tool (for example, `/brainstorm` in Claude Code, or `$brainstorm` in Codex).
To add a command, create a Markdown file in this folder (for example, `brainstorm.md`) with the command definition.


## Contents

* [brainstorm.md](brainstorm.md) - Brainstorm a product idea, problem space, or strategic question with a sharp thinking partner.
