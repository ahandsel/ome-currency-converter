# Notes

Dated research and reference notes for this repository. Each note captures a snapshot of information at a point in time, such as audits, surveys, or working memos.


## Naming

Use the `YYYY-MM-DD-<slug>.md` format so notes sort chronologically and each file states its own date. This format is enforced by the [`file-folder-name-linter`](../skills/file-folder-name-linter/SKILL.md) skill.
